Notas de diseño

🎥

Movimiento de cámara

THREE.CatmullRomCurve3 traza la ruta entre habitaciones. useSpring de Framer Motion controla el timing (800ms, easeInOutCubic). Navbar sincroniza la posición activa.

🃏

Tarjetas flotantes HTML

<Html> de @react-three/drei ancla elementos React al espacio 3D. Animación: translateY -8px → 0 en bucle de 3s. backdrop-blur + white/80 para legibilidad.

📱

Responsive

Mobile: pixel ratio 0.75, rooms en grid 2x3, cámara más cercana. Tablet: 3 columns. Desktop: 5 en línea. Touch events reemplazan scroll wheel en móvil.
